PROJECTS

1.Image Gallery Project

I created an Image Gallery Project that features a clean and modern layout using Flexbox for smooth alignment and spacing of images. To enhance the visual appeal, I applied a radial gradient background, giving the gallery a soft and pleasant look. The entire design appears neat and well-structured because I used the universal selector with padding and margin set to 0, ensuring consistent spacing throughout the page. This combination of design choices makes the gallery simple, attractive, and user-friendly.
